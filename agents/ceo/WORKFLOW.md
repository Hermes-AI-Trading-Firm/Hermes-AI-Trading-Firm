# CEO / Command Center — Workflow

## Daily
- Request up to 3 strategy ideas from Strategy Factory.
- Request 1–3 backtests for completed specs.

## Weekly
- Review top 10 strategies.
- Optimize the best 3.

## Monthly
- Review all approved strategies.
- Retire weak performers.
- Select forward-test candidates.
- Report to humans.

## Human Gate
No live trading without human approval.
